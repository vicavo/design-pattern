This directory is all about design pattern study cases.
1. 观察者模式: 模拟酷米客，观察当前关注的公交的路程状态，观察者分别：终端手机客户，公交调度中心，公交线路日志。
2. 适配器模式：对std::string进行适配和功能扩展，提供更多的string操作。
3. 组合模式：a. 实现一个int类型的数组，数组成员可以是int也可以是数组本身。
             b. 实现一个profile节点访问类，提供通用的操作(叶子节点和树干节点，类比python缩进来控制级别)
4. 装饰者模式： 实现类似王者荣耀游戏中的一个英雄添加装备的情况来提供英雄的伤害等级。装备用来装饰英雄。
5.桥接模式：模拟虚拟驾驶，2个维度，第一个维度各种车(小车，卡车，公交车)，第二个维度道路（水泥路面，柏油路面，沙土路面，泥巴路面）
6. 迭代器模式：实现动态数组ArrayList，并提供访问该数组的迭代器接口。
